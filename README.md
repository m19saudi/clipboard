# Online Clipboard

This is a simple **temporary clipboard** web app. 

- **Frontend** is built using HTML, CSS, and JavaScript.
- **Backend** is built with Node.js and Express.js.

### Features:
- Share text with a unique code.
- Retrieve the text using the code.

### How to Run:
1. Run the **backend** on your server.
2. Use the **frontend** hosted on GitHub Pages or your custom hosting.
